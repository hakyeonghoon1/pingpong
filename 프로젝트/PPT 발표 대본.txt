하진원팀의 하경훈, 김진영, 김지원 입니다.
저희가 개발한 서비스는 실시간 SNS CHAT 기능입니다. 
목차를 설명해드리겠습니다. 
첫번째로는 ~ 입니다.

핑퐁 서비스를 개발하게 된 배경은 카카오톡은 업무와 일상의 경계가 없어 업무 외적인 연락이 많았습니다 그래서 업무에 대한 집중도가 떨어지게 되었고, 노션의 경우 문서공유 부분은 좋았으나, 
실시간 소통에 대한 부분이 아쉬웠습니다.
이런 점들을 보완하고자 데이터 집약적 실시간 애플리케이션인 핑퐁서비스를 개발하게 되었습니다. 
핑퐁이라는 이름의 탄생 배경은 탁구에서의 핑퐁처럼 대화를 원할하게 주고 받을 수 있는 애플리케이션을 제작하기 위함입니다.



다음으로는 저희 핑퐁서비스의 개발과정을 소개해드리겠습니다.

================================================

회원관리 페이지를 소개해 드리겠습니다.
첫번째로 로그인 페이지를 소개해드리겠습니다.
이메일과 패스워드를 입력하여 로그인하는 방식이고 이메일 형식을 체크하는 유효성 검사가 가능합니다. 
회원가입 후 최초 로그인인 경우에는 소속팀이 존재하지 않습니다.
이런 경우에는 웰컴페이지로 이동되며, 그 외에 소속 팀이 있는 경우에는 해당 팀의 단체 채팅방으로 즉시 이동되게 됩니다.
아래에는 각각 회원가입 페이지와 ID찾기 페이지로 이동할 수 있는 버튼이 있습니다. 

회원가입 버튼 클릭시 회원가입 페이지로 이동하게 됩니다.
원하는 이메일 입력시 해당아이디의 중복검사가 시행되며 만약 중복되는 이메일이 있다면 이메일 입력 텍스트박스 아래에 사용중인 Email이라는 문구를 띄우게 됩니다.
여기서 로그인과 마찬가지로 이메일 형식에 맞지 않는 값을 입력 후 회원가입 버튼 클릭시 이메일 형식이 맞지 않는다는 경고 문구를 띄웁니다.
그리고 이메일, 패스워드, 이름, 전화번호의 공백 유효성검사와 비밀번호 일치여부 유효성 검사가 진행됩니다. 이 때 하나라도 요구 조건을 충족하지 않는다면 경고문구가 뜨게 되며 회원가입이 불가능합니다.
이후 모든 요건 충족시 회원 가입이 완료되며 자동으로 로그인 페이지로 이동하게 됩니다.

다음은 아이디 찾기 페이지입니다.
여기서는 사용자의 이름과 사용자의 전화번호를 입력하여 사용자의 이메일 정보를 확인할 수 있는 페이지 입니다.
//====사용자의 이름과 전화번호가 일치하지 않는다면 일치하는 사용자가 없다는 경고문구가 나타나게 되고, 
일치한다면
사용자의 이메일을 조회할 수 있습니다.

다음으로는 회원정보 페이지 입니다. 본인의 프로필 사진과 이메일, 이름, 전화번호, 회사정보가 나타나는 영역입니다. 회원 정보 수정 클릭시 회원 정보 수정화면으로 전환이 되며 
이름과 이메일을 제외한, 프로필 이미지, 휴대번호 , 회사정보 수정이 가능합니다. 파일업로드 기능을 사용하여 이미지 업로드가 가능합니다. 이미지 업로드시 즉시 변경된 이미지가 표시됩니다. 수정완료후 회원정보수정 버튼을 누르게 되면 
회원정보 페이지로 전환이 됩니다.

다음은 웰컴 페이지로서 소속한 팀이 없는 경우 로그인시 이동되는 페이지 입니다. 이 페이지에서는 새로운 팀을 생성하여 프로젝트를 진행할 수 있는 팀의 단체 채팅방으로 이동됩니다. 팀생성시 자동으로 해당팀의 단체 채팅방이 개설됩니다.
초대장을 수락하여 해당 팀에 가입할 수 있습니다. 수락 버튼을 클리하면 팀페이지의 단체 채팅방으로 이동하게 됩니다.

팀 페이지로 이동시 좌측 상단에 현재팀의 이름이 나타나있습니다. 드랍다운박스 클릭시 팀 생성을 할 수 있는 텍스트 박스와 팀 팀리스트가 내려오게 됩니다.
텍스트 박스에 팀명을 입력하고 엔터키를 누르면 팀이 생성되며 해당팀의 단체 채팅방으로 이동하게 됩니다. 그리고 팀리스트의 팀 클릭시 클릭한 팀의 단체 채팅방으로 이동됩니다.

팀페이지로 이동시 팀원 초대가 가능합니다. 
플러스 버튼을 누르게 되면 팀원을 초대할 수 있는 모달 창이 뜨게 됩니다. 여기에는 우리팀에 속해있지 않은 핑퐁 서비스를 이용하는 모든 사용자를 검색할 수 있습니다.
이름을 검색하여 복수의 팀원 초대가 가능하며 초대하기 버튼 클릭시 실시간 비동기 통신을 이용하여 상대방 화면에 초대장 도착 알림과 함게 프로필 화면에 초대장이 나타나게 됩니다.

다음은 채팅방 개설 기능으로서 +버튼을 클릭하면 채팅 상대방을 선택할 수 있는 모달창이 나타나게 됩니다.
팀내에 소속된 팀원을 검색하여 1:1 채팅방을 개설할 수 있습니다. 
채팅방 개설 버튼을 누르면 본인에게는 상대방의 이름으로된 채팅방이 상대방에게는 자신의 이름이 나타난 채팅방이 개설되게 됩니다.
- 버튼을 클릭하여 채팅방 나가기가 가능합니다.

다음은 채팅방 페이지로 해당 대화방 탭을 클릭하여 대화방에 입장할 수 있습니다.
텍스트 박스에 메세지를 입력하여 상대방에게 전송할 수 있습니다. 빈값 입력시 전송 버튼은 작동하지 않고 메시지도 전송이 되지 않습니다.
이 페이지에서 실시간 채팅이 가능하며 드랍다운 클릭시 공지사항등록 기능을 사용할 수 있고, 이를 이용해 공지사항을 등록할 수 있습니다.
공지사항이 등록되면 상대방 화면에서도 실시간으로 공지사항이 등록되거나 변경이 됩니다.
