1G = 첫줄로 이동

dd = 현재 행 잘라내기
yy = 한 줄 복사
(숫자)yy = 현재 행에서 숫자만큼  행 복사
p = 다음 라인에 캐스트
"(저장할 이름 )(행 수) 
"a2yy = a라는 이름으로 2줄 저장
"ap = a라는 이름으로 저장된것 붙여넣기

검색
/(검색어) = 탐색
n = 다음 검색
N = 이전 검색
:!명령어 = 밖에 나가서 하기 귀찮을때, 명령어 한방칠때
:sh = 명령어 여러방
